# Netflix SQL Data Analysis

This project analyzes the Netflix titles dataset using SQL.

Dataset contains more than 8800 movies and TV shows available on Netflix.

## Analysis performed

- Count of Movies vs TV Shows
- Top countries producing Netflix content
- Content distribution by release year
- Most common ratings
- Most frequent genres

## Files

data/netflix_titles.csv → Netflix dataset  
queries.sql → SQL analysis queries
